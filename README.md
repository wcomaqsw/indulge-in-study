使用了requests库，需pip安装。

####  Features： 
-　配置了log日志文件,方便查看信息，也不至于输出太乱 

-　解决了需选择多课程问题 

-　然后就没什么features了，如果需要实现一台机子上抢多个账号的话，需要代理到不同的ip去开进程。不太推荐多线程，容易竞争数据。
